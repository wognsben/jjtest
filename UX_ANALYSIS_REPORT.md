## 📖 About 페이지

### ⚠️ **문제 발견: FirstRite 섹션**

**구조:**
- 부모 컨테이너: 패딩 없음
- ChapterSection: `clamp(1.5rem, 4vw, 3rem)` = 모바일 24px
- 각 섹션: 자체 패딩 `px-6` 또는 `clamp(1.5rem, 4vw, ...)`

**⚠️ 문제 섹션:**

### 1. **FirstRite.tsx** - 과도한 `<br /><br />`

**위치:** `src/components/about/FirstRite.tsx`

**문제:**
```tsx
// 206-209줄
질감은 감정을 닮아 있습니다.<br /><br />
엔카우스틱의 표면은 거칠고 매끄럽고, 때로는 겹쳐집니다.<br /><br />
아이의 마음처럼 여러 결을 가진 그대로.<br /><br />
그래서 이 작업은 감정의 언어를 찾는 첫 연습이 됩니다.
```

**왜 불편한가:**
- `<br /><br />`는 문단 간격이 2줄로 넓어져 **감정적 흐름이 끊김**
- 모바일에서 텍스트가 산만하게 보임
- 감정적 텍스트인데 가독성이 떨어짐

**해결 방법:**
```tsx
// 변경 전
질감은 감정을 닮아 있습니다.<br /><br />
엔카우스틱의 표면은 거칠고 매끄럽고, 때로는 겹쳐집니다.<br /><br />

// 변경 후 (문단으로 분리)
질감은 감정을 닮아 있습니다.
엔카우스틱의 표면은 거칠고 매끄럽고, 때로는 겹쳐집니다.
아이의 마음처럼 여러 결을 가진 그대로.

그래서 이 작업은 감정의 언어를 찾는 첫 연습이 됩니다.
```

**수정 필요 위치:**
- 206-209줄: `<br /><br />` → 문단 분리
- 222-224줄: `<br /><br />` → 문단 분리  
- 237-240줄: `<br /><br />` → 문단 분리

**예상 효과:**
- 감정적 흐름 유지
- 가독성 향상
- 모바일에서 텍스트가 더 자연스러움

## 🔧 권장 수정 사항

### **FirstRite.tsx - `<br /><br />` 제거**

**문제 패턴:**
- `<br /><br />` 사용 → 문단 간격이 과도함

**해결:**
- `<br /><br />`를 제거하고 `<p>` 태그로 문단 분리
- 또는 `<br />`만 사용하여 자연스러운 줄바꿈

**예시:**
```tsx
// 변경 전
<p>
  질감은 감정을 닮아 있습니다.<br /><br />
  엔카우스틱의 표면은 거칠고 매끄럽고, 때로는 겹쳐집니다.<br /><br />
</p>

// 변경 후
<p>질감은 감정을 닮아 있습니다.</p>
<p>엔카우스틱의 표면은 거칠고 매끄럽고, 때로는 겹쳐집니다.</p>
<p>아이의 마음처럼 여러 결을 가진 그대로.</p>
<p>그래서 이 작업은 감정의 언어를 찾는 첫 연습이 됩니다.</p>
```
